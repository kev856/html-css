¿Qué es un em?
-Vale los mismo que el font-size del elemento padre
Si el padre tiene 2px y se coloca 1em, el valor será el mismo, si se coloca 2em medirá 4px, si el elemento padre no tiene definido el font-size, em buscará el siguiente elemento contenedor, si no encuentra ninguno, colocará el que trae el navegador por defecto que son 16px

Y el rem?
-En ves de ir al elemento padre, va al elemento raiz, el cual es html

vw=viewpot width
